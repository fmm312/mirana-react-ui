# **Contribuição**

---

Nos ajude a melhorar esse projeto, implementando novos componentes, fazendo refatorações ou sugerindo melhorias, adoraríamos tê-lo como parte deste projeto!

Veja abaixo como baixar o projeto:

```
$ git clone REPOSITORY_PATH
$ cd oceanic-react-ui
$ npm install ou yarn
$ npm start ou yarn start
```
